# 🌍 Air Quality Prediction & Analysis using Machine Learning

## 📌 Project Overview

This project analyzes global air pollution data and predicts Air Quality Index (AQI) using machine learning techniques. It also includes sensitivity analysis to understand how different pollutants affect AQI.

---

## 🎯 Objectives

* Analyze air pollution data using EDA
* Build a predictive model for AQI
* Understand impact of pollutants like PM2.5, NO2, CO, and Ozone
* Perform what-if analysis to simulate pollution changes

---

## 📊 Dataset

* Global Air Pollution Dataset
* Features include:

  * AQI Value
  * PM2.5 AQI
  * NO2 AQI
  * CO AQI
  * Ozone AQI

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🤖 Models Used

* Linear Regression
* (Optional) Random Forest

---

## 🔍 Key Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* AQI Prediction Model
* What-if Analysis (10% pollutant increase impact)

---

## 🚀 Unique Contribution

* Implemented sensitivity analysis to evaluate how AQI changes when pollutant levels vary
* Identified dominant pollutant affecting AQI (PM2.5)
* Addressed AQI saturation issue (values capped at 500)

---

## 📈 Results

* Achieved high accuracy (R² ≈ 0.97)
* Strong correlation between PM2.5 and AQI
* PM2.5 found to be the most influential pollutant

---

## 🧠 Conclusion

The project demonstrates how machine learning can be used not only for prediction but also for environmental insight and decision-making.
